Set up your workspace
Connect your coding agent to start building personalized skill rules automatically. Follow the steps below to get started.
1
Install the CLI
Install the AutoSkills CLI on the machine you want to connect.
Install command
curl -fsSL https://raw.githubusercontent.com/Royaltyprogram/autoskills-cli/main/scripts/install.sh | sh
2
Create a CLI token
Generate a token to authenticate your CLI with this workspace.
Your CLI token
3
Run setup in your repo
Open a terminal in the repository you want to analyze, then run setup. The CLI will register this device, connect the repo, and start collecting sessions.
Setup command
autoskills setup
