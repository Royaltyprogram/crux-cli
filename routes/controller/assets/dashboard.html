Welcome to AgentOpt
AgentOpt watches how you use your coding agents, then suggests better configurations — one safe change at a time. Approve it, and we'll measure the impact. If it doesn't help, we roll it back automatically.
Install the CLI
Install the CLI on the machine you want to connect. It collects session data in the background.
Install command
curl -fsSL https://raw.githubusercontent.com/Royaltyprogram/aiops/main/scripts/install.sh | sh
Authenticate your CLI
Create a token here, then run the login command in your terminal.
Your CLI token
Click "Create CLI token" above to generate one.
Login command
agentopt login --server http://127.0.0.1:8082
Connect a workspace
Link a local repository so AgentOpt can start observing your workflow and suggesting improvements.
Connect command
agentopt connect --repo-path .
