Welcome to Crux
Crux reads your Codex session traces and explains what the model understood, where it went wrong, and what confused it. Clear analysis instead of a black box.
Install the CLI
Install the CLI on the machine you want to connect. It collects session data in the background.
Install command
curl -fsSL https://raw.githubusercontent.com/Royaltyprogram/aiops/main/scripts/install.sh | sh
Create a CLI token
Generate a token here. In the next step, you'll run setup in the repository you want to analyze.
Your CLI token
Click "Create CLI token" above to generate one.
Run setup
Open a terminal in the repository you want to analyze, then run setup. Crux will prompt for the token, register this device, connect the repo, and upload your local Codex sessions right away.
Setup command
crux setup
